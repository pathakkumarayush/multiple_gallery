# multiple_gallery
Image Gallery Collection
Overview

This project is a creative image gallery collection developed using HTML, CSS, and JavaScript. It contains three different gallery styles with modern UI effects and smooth interactions. The purpose of this project is to showcase different frontend design concepts and animation techniques.

The project includes:

3D Gallery
Circular Gallery
Flex Image Gallery

Each gallery provides a unique visual experience and demonstrates different CSS and JavaScript effects.

Features
1. 3D Gallery
3D rotating image effect
Smooth transitions and animations
Interactive visual appearance
Modern design layout
2. Circular Gallery
Images arranged in a circular pattern
Animated rotation effect
Attractive and creative UI
Responsive structure
3. Flex Image Gallery
Flexible image expansion effect
Smooth hover animations
Dynamic resizing layout
Clean and minimal design
Technologies Used
HTML5
CSS3
JavaScript
Project Purpose

This project was created to practice frontend development concepts such as:

CSS animations
Flexbox layouts
3D transforms
Responsive design
JavaScript DOM manipulation

It can also be used as:

Frontend practice project
UI design reference
Portfolio project
Learning resource for beginners
